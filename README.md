# Afropython Conf 2019

Repositório para o site da Afropython Conf 2019

## Tecnologias

- HTML5
- CSS
- Github Pages

## Decisões Técnicas

É uma página única feita apenas com HTML5 e CSS onde se encontra as principais informações do evento.

O CSS foi dividido entre _estrutura_ e _tipografia_. Todo o css relacionado a estrutura e posicionamento do site está em `index-structure.css` já o css relacionando a tipografia, estilos e cores está em `index-typography.css`. Dessa forma, quando formos dar cores ao site, basta focar apenas em um arquivo css, sem medo de cagar gerar problemas na estrutura da página.

## Como colaborar

- Clonando o repositório:

`https://github.com/AfroPython/afropython-conf-2019.git`

- Fazendo alterações de HTML em `index.html` e de CSS nos arquivos `index-structure.css` para mudanças em estrutura e `index-typography.css` para mudanças de estilo.
